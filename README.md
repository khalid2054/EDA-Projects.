# EDA Projects 📊

A collection of Exploratory Data Analysis projects built from scratch as part of my self-taught ML engineering journey.

---

## 📁 Projects

### 1. Netflix EDA
- **Files:** `netflix(eda).py`, `netflix_titles.csv`
- **Goal:** Analyze Netflix content library — movies vs shows, top genres, release trends, country-wise distribution

---

### 2. Zomato EDA
- **Files:** `zomato_eda.py`, `zomato-data-set-analysis-visualization.ipynb`, `Zomato-data-.csv`
- **Goal:** Analyze restaurant data — ratings, cuisines, cost distribution, online ordering trends

---

### 3. Insurance EDA
- **Files:** `insurance(EDA).py`, `insurance.csv`
- **Goal:** Analyze insurance charges — impact of age, BMI, smoking status, and region on charges

---

### 4. Video Game Sales EDA
- **File:** `sale(eda).py`
- **Goal:** Analyze global video game sales by genre, platform, publisher, and region
